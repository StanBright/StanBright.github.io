LibHunt

2016

Visit website


Find the software you need. The goal is to help you find the software and libraries you need to get your job done. All lists have been crafted by many experts from relevant GitHub communities.

LibHunt consists of many sub-websites that are dedicated to different programming languages or tech. For example Awesome Rbuy, Awesome Python and many others.

All “awesome” sub-sites in LibHunt have a manually currated weekly newsletter. The total number of subscribers as of Sep 2020 is a bit more than 15,000.
